Portfolio Theme Installation Guide

This guide will walk you through the process of installing and deploying a TemplateMonster portfolio theme in a Next.js project. By following these steps, you'll be able to set up your portfolio website using the theme.

Prerequisites

Before you begin, ensure that you have the following installed on your system:

  • Node.js: Make sure you have Node.js installed. You can download it from the official website and follow the installation instructions for your operating system.

Installation Steps

  • Download the portfolio theme :
    • Download the purchased portfolio theme from the TemplateMonster website link.
    • Save the downloaded theme to a convenient location on your computer.
  • Unzip the theme package:
    • Extract the downloaded theme to a folder of your choice.
  • Install project dependencies
    • Open your terminal or command prompt.
    • Navigate to the root directory of the unzipped theme package.
    • Run the following command to install the project dependencies:

      npm install

  • Run the development server:
    • After the dependencies are installed, start the Next.js development server by running the following command:

      npm run dev

    • The development server will start, and you can access your Next.js project with the theme applied by opening a web browser and visiting http://localhost:3000.
  • Customize and configure the theme:
    • Project file structure:
    • Depending on the theme, you may need to configure certain settings or provide additional information.
    • Replace data with your data in data.js file.
    • Set NEXT_PUBLIC_SHOW_THEME_OPTION="false" for hiding color change option in .env file
    • Set primary color (rgb format) to –color-primary variable in the app/global.css file.
    • Consult the theme's documentation for any required configuration steps.
    • Update the necessary configuration files within your Next.js project based on the instructions provided.
  • Build the project:
    • Once you have customized the theme and are satisfied with the changes, you can build the project for production deployment.

      npm run build

    • The build process will generate an optimized and minified version of your project in the ./out directory.
  • Deploy the project:
    • Choose a server or hosting provider to deploy your Next.js project.
    • Upload the contents of the ./out directory to your server or hosting provider.
    • Follow the instructions provided by your server or hosting provider to deploy your Next.js project.
  • Verify the deployed website:
    • Once the project is deployed, open a web browser and visit your website's URL.
    • Ensure that the deployed website functions as expected and that the theme is applied correctly.

Congratulations! You have successfully installed and deployed your TemplateMonster portfolio theme in a Next.js project. You can now start customizing and adding your own content to create an impressive portfolio website.
